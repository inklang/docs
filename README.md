# inklang Documentation

Documentation for [inklang](https://github.com/inklang/ink), a compiled scripting language for the modern VM.

## Quick Start

```bash
npm install
npm start
```

## About inklang

inklang is a compiled language targeting a register-based bytecode VM with SSA optimizations. It features clean syntax, first-class functions, and a powerful package manager.
